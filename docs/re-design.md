# 포트폴리오 페이지 - 레트로 Windows OS 스타일 리디자인

## 컨셉

Windows 95/98 데스크톱 환경을 재현한 인터랙티브 포트폴리오.
사용자가 실제 OS를 조작하는 듯한 경험을 제공한다.

---

## 데스크톱 환경

### 바탕화면

- Windows 98 클래식 틸(teal) 배경 또는 커스텀 배경
- 픽셀 아이콘들이 좌측 상단부터 세로로 정렬
- 우클릭 시 컨텍스트 메뉴 (배경 변경, 새로고침 등)

### 작업표시줄 (하단 고정)

- [시작] 버튼 + 로고
- 열린 윈도우 목록 (탭 형태)
- 시스템 트레이 (현재 시간, 방문자 수, 볼륨 아이콘 등)
  - 방문자 수: `services/view.tsx`의 `getTodayViewCount()`, `getAllViewCount()` 재활용
  - 표시 형식: `👁 오늘: 42 / 누적: 1,234`
  - `utils/date.ts` 활용하여 시스템 트레이 시계 구현

### 시작 메뉴

- 클릭 시 위로 펼쳐지는 메뉴
- 각 섹션으로의 바로가기
- "시스템 종료" → 외부 링크(GitHub, 블로그 등)로 이동

## 윈도우 시스템 (핵심)

### 공통 윈도우 컴포넌트

- 타이틀바: 아이콘 + 제목 + 최소화/최대화/닫기 버튼
- 메뉴바: 파일 | 편집 | 보기 | 도움말 (장식용 or 기능 연결)
- 콘텐츠 영역: 섹션별 다른 레이아웃
- 상태바 (하단): 부가 정보 표시
- 레트로 스타일 스크롤바

### 윈도우 인터랙션

- 드래그로 위치 이동
- 리사이즈 (모서리/변 드래그)
- 최소화 → 작업표시줄로 내려감
- 최대화 → 전체화면 (작업표시줄 제외)
- 닫기 → 윈도우 제거
- 윈도우 클릭 시 z-index 최상위로 (포커스)
- 여러 윈도우 동시 열기 가능

## 섹션별 윈도우 콘텐츠

### 1. 자기소개.txt (메모장)

- 메모장 UI 재현
- 모노스페이스 폰트로 자기소개 텍스트
- `aboutMeModal/Profile.tsx`의 기본 프로필 정보 재활용 (이름, 연락처, 학력 등)
- 메뉴: 파일 > 인쇄 → PDF 이력서 다운로드 (기존 Notion 경력기술서 링크 대체)

### 2. 내 프로젝트 (파일 탐색기)

- 좌측: 폴더 트리 (카테고리별)
- 우측: 파일/폴더 목록 (아이콘뷰 / 리스트뷰 토글)
- 주소창에 경로 표시: C:\Projects\...
- 프로젝트 더블클릭 → 상세 윈도우 열림
  - 프로젝트 설명, 스크린샷, 기술스택 태그, 라이브 링크/GitHub 링크

### 3. 기술스택.exe (시스템 정보)

- 시스템 속성 창 형태
- 탭 구조: Frontend | Backend | Tools | etc.
- 각 기술을 프로그레스바 또는 시스템 사양 테이블로 표시
- "프로세서: React 18 / 메모리: TypeScript 5년" 같은 유머 요소

### 4. 메일 보내기 (Outlook Express)

- 받는사람: 본인 이메일 (고정)
- 제목: 입력 필드
- 본문: 텍스트 에어리어
- [보내기] 버튼 → `mailto:` 링크 생성 후 기기 기본 메일 앱 실행
  - `mailto:{내 이메일}?subject={제목}&body={본문}` 형태로 인코딩
  - `window.location.href` 또는 `<a href="mailto:...">` 로 실행
- 별도 서버/이메일 서비스 불필요, 클라이언트 단독 처리

### 5. 내 경력 (파일 탐색기)

- `aboutMeModal/Carrer.tsx` 콘텐츠 재활용
- 파일 탐색기 형태로 경력 사항 나열
- 회사별 폴더 구조로 경력 기간, 담당 업무, 성과 표시

### 6. 인터넷 (Internet Explorer)

- 주소창 + 이동/뒤로/앞으로 버튼
- 즐겨찾기 바: GitHub, Blog, LinkedIn 등
- 콘텐츠 영역: 링크 목록 or 간단한 페이지

## 부팅 시퀀스 (진입 애니메이션)

- BIOS POST 화면 → Windows 로고 + 프로그레스바 → 바탕화면 진입
- 최초 방문 시 1회 재생, 이후 스킵 가능
- "아무 키나 누르세요" 또는 자동 진행

---

## 반응형 & 모바일 (필수)

### 데스크톱 (1024px+)

- 풀 OS 경험: 드래그, 리사이즈, 멀티 윈도우 모두 지원

### 태블릿 (768px ~ 1023px)

- 윈도우 드래그 유지, 리사이즈 제한
- 아이콘 크기/간격 확대 (터치 대응)
- 윈도우 열릴 때 화면의 80~90% 차지

### 모바일 (~ 767px)

- 바탕화면: 앱 런처 스타일 그리드 (스마트폰 홈화면 느낌)
- 아이콘 탭 → 윈도우가 풀스크린으로 열림
- 드래그/리사이즈 비활성화
- 작업표시줄 → 하단 네비게이션 바로 변환
- 뒤로가기 = 윈도우 닫기
- 핵심: "레트로 OS 감성은 유지하되 모바일 UX는 네이티브앱처럼"
